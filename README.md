# 🚗 Vehicle Sales Intelligence Dashboard | Power BI Analytics Project

## 📌 Project Overview

This project focuses on transforming raw automotive sales data into an executive-level business intelligence solution using **Power BI**.

The dashboard was developed to simulate how automotive leadership teams monitor:

- Regional sales performance
- Vehicle demand patterns
- Fuel type adoption trends
- Dealer intelligence
- Product performance
- Market expansion opportunities

Rather than building a basic visualization project, the objective was to create a stakeholder-oriented analytics solution capable of supporting strategic business decisions.

---

# 🎯 Business Problem

Automotive companies generate massive volumes of transactional sales data across multiple regions, vehicle categories, and dealer networks.

However, leadership teams often face challenges such as:

- Fragmented reporting systems
- Lack of centralized KPI visibility
- Slow business decision-making
- Limited trend analysis capabilities
- Difficulty identifying high-performing markets and products

This project addresses those challenges by building a centralized interactive Power BI dashboard for executive analytics.

# 📈 Key Business Insights

## 1. Sales Trend Intelligence
- Tracks year-over-year sales growth
- Identifies demand fluctuations
- Supports strategic forecasting discussions

## 2. Regional Performance Analysis
- Evaluates market-wise sales performance
- Identifies high-growth regions
- Highlights underperforming territories

## 3. Vehicle Model Performance
- Ranks top-performing vehicle models
- Identifies declining product categories
- Supports portfolio optimization decisions

## 4. Fuel Type Adoption Trends
Analyzes customer demand across:
- Electric Vehicles
- Hybrid Vehicles
- Petrol Vehicles
- Diesel Vehicles

Supports future mobility and EV strategy discussions.

## 5. Transmission Preference Analysis
Measures customer preference between:
- Automatic
- Manual

Supports production and inventory planning.
